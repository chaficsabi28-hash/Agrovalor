# AgroValor Industrie

AgroValor Industrie est une entreprise agro-industrielle spécialisée dans la transformation du manioc en produits alimentaires à forte valeur ajoutée : farine, gari, tapioca et amidon.

Ce dépôt contient le code source du site web officiel d’AgroValor Industrie.

## Contenu
- index.html : page principale du site
- images/ : images illustratives
- styles/ : styles intégrés

## Objectif
Présenter l’entreprise, ses produits, son impact et faciliter la prise de contact avec des partenaires, distributeurs et investisseurs.
